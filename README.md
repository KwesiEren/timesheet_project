## 🚀 Quick Start

### Prerequisites
- Flutter SDK (v3.5.4+)
- Node.js (v18+)
- PostgreSQL (or Supabase)

### Running the App
1. Clone the repository.
2. Setup the backend (see [Backend Docs](docs/backend_documentation.md)).
3. Run `flutter pub get` in the root directory.
4. Run `flutter run`.

## 🛠 Tech Stack
- **Frontend**: Flutter, GetX, Dio, Get Storage.
- **Backend**: Node.js, Express, PostgreSQL, JWT.
- **Infrastructure**: Supabase (recommended for DB).
